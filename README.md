# To-Do List CLI Application

A simple command-line to-do list manager built with Python.

## Features

- **Add tasks** — enter a description to add it to your list
- **View tasks** — see all tasks with numbered indices
- **Delete tasks** — remove a task by its number
- **Input validation** — handles empty input, invalid numbers, and out-of-range selections
- **Error handling** — uses try/except/else/finally throughout

## How to Run

```bash
python todo.py
```

## Menu Options

| Option | Action               |
|--------|----------------------|
| 1      | Add a new task       |
| 2      | View all tasks       |
| 3      | Delete a task        |
| 4      | Quit the application |

## Requirements

- Python 3.6 or higher
- No external dependencies
